## Installation

```bash
npm install @rapid/babel-plugin-jsx --save-dev
```

## Usage

```json
{
  "plugins": ["@rapid/babel-plugin-jsx"]
}
```

## Features

- **Lazy children** - Children are wrapped in functions for fine-grained reactivity
- **Automatic transforms** - No manual wrapping needed
- **Compatible** - Works with Vite, Webpack, and other bundlers
